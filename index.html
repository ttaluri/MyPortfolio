Project 1: ToDo List Web Application (JavaScript, React, Bootstrap, Node.js)

Developed a dynamic To-Do List Web Application using React and Bootstrap that allows users to create, edit, and organize tasks. The application includes features such as moving tasks between lists and color-coded logic to highlight tasks based on their due date proximity. To ensure data reliability and seamless recovery, I refactored the backend API to serialize and persist tasks in a JSON file. By leveraging React’s state management with `useState` and implementing PUT/POST methods, I enhanced the user experience with responsive forms, enabling real-time task updates and seamless synchronization with the backend.
